مدير ملفات بتصميم المواد مفتوح المصدر.

المميزات:
- مفنوح المصدر: خفيف الوزن ونظيف وآمن.
- التصميم متعدد الأبعاد: يتبع إرشادات التصميم متعدد الأبعاد ، مع الاهتمام بالتفاصيل.
- فتات الخبز: تنقل في نظام الملفات بسهولة.
- دعم ميزة الجذر: عرض وإدارة الملفات مع الوصول إلى الجذر.
- دعم الأرشفة: عرض واستخراج وإنشاء ملفات مضغوطة مشتركة.ِ
- دعم NAS: عرض وإدارة الملفات على خوادم FTP و SFTP و SMB.
- السمات: ألوان واجهة مستخدم قابلة للتخصيص ، بالإضافة إلى الوضع الليلي مع اللون الأسود الحقيقي الاختياري.
- مدرك لنظام Linux: يعرف الروابط الرمزية وأذونات الملفات وسياق SELinux.
- قوي: يستخدم استدعاءات نظام Linux تحت الغطاء ، وليس محلل ls آخر.
- مُنفَّذ جيدًا: مبني على الأشياء الصحيحة ، بما في ذلك Java NIO2 File API و LiveData.
